Comment utiliser EFREIMOTION

Testez dès maintenant gratuitement notre nouvelle application EFREIMOTION !

  1. Télécharger l'application
  2. Extraction de l'archive :
    1. Faites un clic droit sur le fichier ZIP et sélectionnez l'option "Extraire tout" dans le menu contextuel qui apparaît.
    2. Choisissez l'emplacement où vous souhaitez extraire les fichiers de l'archive. Il peut s'agir d'un dossier spécifique ou simplement du bureau.
    3. Cliquez sur le bouton "Extraire" pour extraire les fichiers de l'archive vers l'emplacement sélectionné.
  3. Lancement de l'application :
    1. Ouvrez le dossier extrait de l'archive.
    2. Recherchez le fichier EFREIMOTION.exe
    3. Faites un double-clic sur le fichier EFREIMOTION.exe pour lancer l'application.
  4. Vous avez maintenant accès à l'application EFREIMOTION
